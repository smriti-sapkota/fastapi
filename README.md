# FastAPI Blog API

This is a backend API project built using **FastAPI** as part of my learning journey in backend development.  
The project implements basic social media features such as user authentication, posts, and voting.

## Features
- User registration and login using JWT authentication
- Create, read, update, and delete posts
- Vote/like system for posts
- Password hashing for security
- Authorization so users can modify only their own posts
- PostgreSQL database integration
- Database migrations using Alembic

## Technologies Used
- **FastAPI**
- **PostgreSQL (Neon)**
- **SQLModel**
- **OAuth2 & JWT**
- **Alembic**
- **Render (Deployment)**
